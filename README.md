Wordle Game: Play Wordle in your terminal! This is a simple Python version of the classic Wordle game. You can play this game either by running it with Python or by using a pre-built executable. Option 1: Run with Python. Make sure you have Python 3 installed on your computer. Clone this repository by running git clone https://github.com/samiyamenon/Wordle.git. Go into the project folder with cd Wordle. Install dependencies if any exist by running pip3 install -r requirements.txt. Run the game with python3 main.py. The game will run in your terminal; follow the on-screen instructions to play. Option 2: Use a Pre-Built Executable. Go to the Releases tab of this repository. Download the executable for your system. For Mac, download Wordle and run it in Terminal with ./Wordle. For Windows, download Wordle.exe and double-click to run. Mac users: if you see a warning like “Apple could not verify …”, right-click the executable, select Open, then click Open in the warning dialog. The game will now run in Terminal. Tips: The game is text-based, so it runs in Terminal (macOS) or Command Prompt (Windows). Make sure your terminal is large enough to see the board properly. To reset your progress, close and reopen the game. Enjoy playing Wordle!
