Wordle Game: Play Wordle in your terminal! This is a simple Python version of the classic Wordle game. You can play this game either by running it with Python. Make sure you have Python 3 installed on your computer. Clone this repository by running git clone https://github.com/samiyamenon/Wordle.git. Go into the project folder with cd Wordle. Install dependencies if any exist by running pip3 install -r requirements.txt. Run the game with python3 Wordle.py. The game will run in your terminal; follow the on-screen instructions to play.
